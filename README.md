# Portfólio — Vanessa Santos

Site estático em HTML + React (via Babel CDN). Sem build, sem dependências locais.

## Arquivos

- `index.html` — página principal (este é o arquivo servido na raiz)
- `cases.jsx` — dados e componentes dos cases
- `assets/` — imagens dos cases

## Como publicar no GitHub Pages

1. Crie um repositório novo (público) em github.com — sugestão de nome: `portfolio`
2. Faça upload de **todos os arquivos desta pasta** (mantendo a estrutura)
3. Vá em **Settings → Pages**
4. Em **Source**, escolha **Deploy from a branch** → branch `main` → folder `/ (root)` → Save
5. Aguarde ~1 minuto. O endereço será `https://SEU-USUARIO.github.io/portfolio/`

### Quer URL sem o `/portfolio/` no final?

Nomeie o repositório como `SEU-USUARIO.github.io` (exatamente assim). Aí o site fica em `https://SEU-USUARIO.github.io/`.

### Domínio próprio (opcional)

Em **Settings → Pages → Custom domain**, cole o domínio (ex.: `vanessasantos.com.br`) e configure os registros DNS conforme a documentação do GitHub.
